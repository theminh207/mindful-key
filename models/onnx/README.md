# models/onnx

Chưa có model file (`.onnx`). Định dạng trung gian dùng cho **Windows** (ONNX Runtime/DirectML)
và làm nguồn convert sang Core ML (macOS/iOS) hoặc TFLite (Android).

Trạng thái hiện tại: SPEC — chưa code, chưa train/export. Xem
`../../docs/SEND-RISK-MODEL-SPEC.md` (thay lexicon bằng PhoBERT ONNX cho send-risk 0–1).
