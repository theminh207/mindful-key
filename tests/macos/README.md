# tests/macos/

**Sở hữu:** đội macOS — test riêng vỏ macOS (`platforms/apple/macos/`). Không đụng `tests/core/`.

**Chạy:** `make test-macos` (hoặc `make test` chạy chung cả core/macos/ios).

**Trạng thái hiện tại:** trống — vỏ macOS hiện chưa có test tự động riêng (nghiệm thu thủ công theo `bmad-output/macos/TEST-PLAN-macos-control-panel.md`). `make test-macos` tạm thời no-op cho tới khi có test thật; đừng biến no-op thành "đã test".
