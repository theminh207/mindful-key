# apple/shared

Chưa mở. Chỗ dành cho code Objective-C/Swift dùng chung giữa macOS và iOS khi nhánh
iOS bắt đầu (ví dụ: `BrandColors`, model schema, wrapper `core/mood`) — hiện `core/engine`
+ `core/mood` (C++ thuần) đã đóng vai trò dùng-chung ở tầng thấp hơn, nên thư mục này
chỉ cần khi có code Objective-C/Swift thật sự lặp lại giữa 2 vỏ Apple.

Xem thứ tự lộ trình nền tảng: `../README.md`.
