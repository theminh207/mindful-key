## Handoff

Giao cho công cụ code mỗi đợt: tên nhánh worktree + file scope từng story + thứ tự trộn trên. File
story vẫn là nguồn ngữ cảnh chuẩn; mục **Acceptance Criteria / Dev Notes / Testing là LOCKED**,
không sửa khi code. Đường thực thi gợi ý: qua `mindful-keyboard-harness` (platform-shell-agent cho
1.1/1.3/1.4/1.6, mood-layer-agent cho 1.2/1.5).

> **Lưu ý bối cảnh dự án:** repo chính `is a git repository: false` ở phiên khảo sát — nếu chưa
> `git init`, mô hình worktree/nhánh ở trên chỉ áp dụng SAU khi khởi tạo git. Với 1 người làm, có
> thể chạy tuần tự theo đúng thứ tự đợt mà không cần worktree — kế hoạch này vẫn đúng, chỉ là bỏ
> phần cô lập nhánh.
