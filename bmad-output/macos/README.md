# bmad-output/macos/

**Sở hữu:** đội macOS. Được sửa tự do mọi file trong thư mục này (epics, story, design, test-plan, sprint-status, dependency-graph...) mà không cần đội iOS đồng ý.

**Được chứa:** kế hoạch/spec/story chỉ liên quan tới vỏ macOS (`platforms/apple/macos/`) — Epic 1 "Bảng điều khiển macOS" và mọi thứ đi kèm (DESIGN/EXPERIENCE/TEST-PLAN/brainstorm, `stories/`, `context/sharding-context.md`, `sprint-status.yaml`, `dependency-graph.json`, `waves.json`, `parallelization-plan.md`).

**Không chứa:** quyết định ảnh hưởng tới `core/` dùng chung hoặc đội iOS — việc đó ghi vào `bmad-output/_shared/` (kèm 1 dòng tham chiếu ở đây nếu cần).
