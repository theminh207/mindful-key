# bmad-output/ios/

**Sở hữu:** đội iOS. Được sửa tự do mọi file trong thư mục này mà không cần đội macOS đồng ý.

**Được chứa:** kế hoạch/spec/story chỉ liên quan tới vỏ iOS (`platforms/apple/ios/` khi bắt đầu code) — epics, story, design, test-plan riêng của đội này.

**Trạng thái hiện tại:** trống — chưa có việc iOS nào được lên kế hoạch. Khi đội iOS bắt đầu, tạo cấu trúc tương tự `bmad-output/macos/` (epics.md, stories/, v.v.).

**Không chứa:** quyết định ảnh hưởng tới `core/` dùng chung hoặc đội macOS — việc đó ghi vào `bmad-output/_shared/`.
